 ## AI Study Buddy

AI Study Buddy is a web-based application designed to assist students in learning and revision. It allows users to explain academic topics, summarize notes, and generate quizzes using an AI language model through a simple and interactive interface.

## Technologies Used

Python

Streamlit

Groq LLM API

GitHub

Streamlit Community Cloud

## How to Run Locally
Clone the repository
Install dependencies:
pip install -r requirements.txt
Add your API key in a .env file:
GROQ_API_KEY=your_api_key_here

## Run the application:
streamlit run app.py

## Deployment
The application is deployed using Streamlit Community Cloud and can be accessed through the public deployment link.
